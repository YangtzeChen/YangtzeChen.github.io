# 版本修改说明

## V3.1.0 — 2026-03-24

### ✨ CMS 后台 UI 统一化与安全发布 (CMS UI Unification & Secure Publishing)

- **后台卡片 UI 同步**：成功将前台的 `post-card` 高级排版设计移植到后台文章列表。统一了响应式阶梯式字体（Tiered Typography）、渐变遮罩以及右对齐的日期显示。
- **“阅后即焚”安全认证 (Read & Burn Auth)**：引入了基于 Web Crypto API 的 **AES-GCM (256-bit)** 加密层。GitHub Token 仅以密文形式存于 `sessionStorage`，关闭浏览器标签页即刻物理销毁，确保 Token 安全不落地。
- **GitHub API 直接发布系统**：彻底打通了 CMS 与 GitHub 仓库的直连。现在“发布”和“保存草稿”功能会直接通过 REST API 将 Markdown 文件与全局索引推送到仓库，实现真正意义上的在线闭环管理。
- **预设封面色彩系统**：为无封面图的文章引入了首选预设色方案，确保每张卡片在创建之初都具备极佳的视觉表现力。
- **后台乳白色排版优化**：将管理列表中的卡片文字统一优化为乳白色调，配合微小的文字阴影，提升了在各种色彩背景下的阅读质感。

---

### ✨ 封面选择器全面升级 (Cover Picker Enhancement)

- **三分栏等比布局**：优化封面选择器 UI，确保三个预览/选择区域的空间均等分配，提升视觉平衡感。
- **Aero 毛玻璃质感颜色预览**：为颜色预览区引入前台同款“aero”风格的磨砂质感渐变效果，颜色值与后台取色器联动。
- **实时图片预览**：在第三分栏中实时展示所选图片的预览结果（支持本地图片、画廊图片或外部 URL 链接）。
- **图色互斥机制**：实现颜色与图片的严格互斥逻辑——选择一者将自动清除另一者，并在上方预览区即时给出反馈展示。
